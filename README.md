### Miljövariabler

Flera av lektionerna kräver att du sätter upp miljövariabler för att dem skall fungera. Gör detta genom att skapa en fil som heter `.env`. Ge en följande variabler: 

```
PORT=4321
JWT_TOKEN=sghsagw8htonsdf
```
